Run_Spicemix

This tutorial demonstrates how to run the SpiceMix pipeline on 10x Visium spatial transcriptomics data and generate clustering results.

Required Inputs

  • 10x Visium raw_feature_bc_matrix directory
  • 10x Visium tissue_positions_list.csv
  • High-resolution tissue image (.tif)
  • Python scripts: make_spicemix_files.py, spicemix.py, spicemix_LIBD.py

Step 1: Preprocess Data for SpiceMix

Run the following command to convert your 10x Visium data into the format required by SpiceMix:

python3 make_spicemix_files.py

This script will:

  • Load and normalize gene expression data
  • Extract pixel coordinates from the tissue positions file
  • Generate the following files in a files/ subdirectory:
    • expression_SAMPLEID.txt
    • coordinates_SAMPLEID.txt
    • neighborhood_SAMPLEID.txt
    • genes_SAMPLEID.txt
    • meta_SAMPLEID.csv

Step 2: Run the SpiceMix Model

Execute the appropriate SpiceMix runner script:

python3 spicemix.py

Or, for LIBD datasets:

python3 spicemix_LIBD.py [dataset_index]

This script will:

  • Load preprocessed input files
  • Initialize the model using Louvain clustering
  • Run 200 iterations of parameter optimization
  • Save results to: results/SpiceMix.h5
